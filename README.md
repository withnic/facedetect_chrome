# facedetect_chrome

* chrome://flags/#enable-experimental-web-platform-features

* enableにして、画面下のRefresh

## Chrome の起動

localのMacで確認する場合

```
open -a Google\ Chrome --args -allow-file-access-from-files
```

## 動作

* index.htmlを開いてください。
* 画像のパスを書き換えてください。
